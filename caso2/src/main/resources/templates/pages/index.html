Acerca del Sistema

Esta plataforma académica permite gestionar usuarios del sistema según sus roles: Administradores, Profesores y Estudiantes. Cada rol tiene permisos específicos para acceder a diferentes secciones del sistema.
Todo corresponde al caso #2 del curso de Desarrollo de Aplicaciones Web y patrones.

  • ADMIN: Puede gestionar usuarios y roles completos.
  • PROFESOR: Accede a reportes y consultas avanzadas.
  • ESTUDIANTE: Puede ver únicamente su perfil personal.
